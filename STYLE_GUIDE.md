# Guía de Estilo: Orientación Vocacional Bolivia (Gamificada)

Este documento define la identidad visual de la plataforma, inspirada en un diseño moderno, dinámico y tecnológico.

## 1. Concepto Visual
- **Gamificación:** Uso de elementos visuales interactivos, barras de progreso y colores vibrantes para mantener el interés del bachiller.
- **Modernidad:** Tipografía geométrica y bordes muy redondeados.
- **Contexto Regional:** Colores verdes que evocan la agroindustria y el dinamismo de Santa Cruz, Montero y Warnes.

## 2. Paleta de Colores

### Colores Principales
- **Primario (Acción):** `#13ec6d` (Verde Eléctrico) - Usado para botones principales y progreso.
- **Fondo Oscuro:** `#102218` (Verde Bosque Profundo) - Para el modo oscuro.
- **Superficie Oscura:** `#162e21` (Verde Musgo) - Para tarjetas y contenedores en modo oscuro.
- **Fondo Claro:** `#f6f8f7` (Gris Niebla) - Para el modo claro.

### Colores de Acento (Categorías)
- **Logística:** Azul (#3b82f6)
- **Ingeniería:** Naranja (#f97316)
- **Creatividad:** Púrpura (#a855f7)
- **Ventas/Social:** Esmeralda (#10b981)

## 3. Tipografía
- **Fuente Principal:** `Lexend`
- **Títulos:** Bold (700) o ExtraBold, con interletrado ligeramente ajustado.
- **Cuerpo:** Regular (400) o Medium (500) para legibilidad máxima en móviles.

## 4. Componentes y Formas
- **Bordes:** Radio de `1rem` (16px) como estándar, llegando a `full` para botones tipo píldora.
- **Botones:** Efectos de escala al presionar (95%) y sombras suaves.
- **Tarjetas:** Fondo sólido con bordes sutiles o sombras difusas.

## 5. Iconografía
- **Estilo:** Material Symbols Outlined.
- **Implementación:** Iconos dentro de círculos de colores pasteles o con fondos de baja opacidad.
